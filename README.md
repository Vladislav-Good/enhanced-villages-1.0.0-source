# Enhanced Villages Mod (Улучшенные Деревни)

Мод для Minecraft 1.20.1 на Fabric. Добавляет богатое содержание в деревни.

## Что добавляет мод

### 6 Новых профессий жителей
| Профессия | Рабочий стол | Что продаёт |
|-----------|-------------|-------------|
| **Алхимик** | Алхимический котёл | Зелья, ингредиенты, монеты деревни |
| **Торговец** | Рыночный прилавок | Редкие товары, Тотем бессмертия, Элитры |
| **Мастер-кузнец** | Кузнечная наковальня | Алмазное снаряж., зачарованные предметы |
| **Проводник каравана** | Знак каравана | Карты, сёдла, конская броня |
| **Смотритель заданий** | Доска заданий | Свитки заданий, принимает ресурсы |
| **Таверщик** | Сундук торговца | Еда, Золотые яблоки, Зач. Золотые яблоки |

### 3 Новых существа
- **Стражник деревни** — атакует зомби, скелетов, разбойников; носит железную броню
- **Торговец каравана** — путешествующий торговец с уникальными товарами
- **Житель с заданием** — выдаёт задания и награды

### 6 Новых блоков
- **Рыночный прилавок** — центр рынка деревни
- **Доска заданий** — берите задания, нажав ПКМ
- **Знак каравана** — место остановки каравана
- **Кузнечная наковальня** — расширенная кузница
- **Алхимический котёл** — лаборатория алхимика
- **Сундук торговца** — хранилище товаров

### 5 Новых предметов
- **Монета деревни** — особая валюта для торговли
- **Торговый контракт** — даёт скидку 20% у торговцев
- **Карта каравана** — показывает маршрут каравана
- **Медальон стражника** — знак уважения деревни
- **Свиток задания** — содержит ваше активное задание

### Система заданий
1. Найдите **Доску заданий** или **Жителя с заданием**
2. Нажмите ПКМ, чтобы получить свиток задания
3. Выполните задание (собрать ресурсы, убить монстров)
4. Вернитесь к жителю — он проверит выполнение
5. Получите **монеты деревни** и **100 опыта**

### Типы заданий
- Собрать ресурсы (пшеницу, железо, алмазы...)
- Убить монстров (зомби, скелетов, разбойников...)
- Доставить товар соседям
- Исследовать пещеры

## Установка

### Требования
- Minecraft **1.20.1**
- **Fabric Loader** 0.15.11+
- **Fabric API** 0.92.2+

### Шаги установки
1. Установите [Fabric Loader](https://fabricmc.net/use/)
2. Скачайте [Fabric API](https://modrinth.com/mod/fabric-api)
3. Поместите `enhanced-villages-1.0.0.jar` в папку `mods/`
4. Запустите Minecraft!

## Сборка из исходников

```bash
# Клонируйте репозиторий
git clone <url>
cd enhanced-villages

# Соберите мод
./gradlew build

# JAR-файл будет в build/libs/
```
